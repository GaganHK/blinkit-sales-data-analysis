# Blinkit Sales Data Analysis

## 📌 Project Overview
This project analyzes Blinkit sales data to identify revenue trends, product performance, and outlet efficiency. The objective was to clean the dataset, calculate key KPIs, and generate actionable business insights using SQL, Excel, and Power BI.

---

## 🛠 Tools Used
- SQL (Data cleaning & analysis)
- Microsoft Excel (Data transformation & pivot analysis)
- Power BI (Interactive dashboard creation)

---

## 📊 Key KPIs Analyzed
- Total Revenue
- Average Sales per Order
- Number of Orders
- Average Customer Rating
- Sales by Item Type
- Sales by Outlet Type
- Sales by Outlet Size
- Percentage Contribution by Outlet Size

---

## 🔎 Key Analysis Performed

### 1️⃣ Data Cleaning
- Standardized inconsistent values in `Item_Fat_Content`
- Verified cleaned categories using DISTINCT queries

### 2️⃣ Sales Analysis
- Calculated total revenue (in millions)
- Computed average sales and order count
- Analyzed category-wise and item-type sales
- Performed outlet-wise sales comparison using PIVOT
- Calculated percentage sales contribution using window functions

---

## 📈 Key Insights
- Certain item categories contributed significantly higher revenue.
- Outlet size and location had a direct impact on total sales performance.
- Top-performing outlet types generated higher average sales and ratings.
- A small percentage of items contributed to a large share of revenue (Pareto effect).
